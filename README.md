# Tesla Custom Home Assistant Integration

This allows you to integrate your Tesla vehicle into Home Assistant using an OAuth token for the Tesla APIs.

## Installation

### HACS
1. Open HACS in Home Assistant.
2. Click on Integrations.
3. Add this repository as a custom repository.
4. Download the integration.
5. Restart Home Assistant.

### Configuration
Go to **Settings -> Devices & Services -> Add Integration** and search for "Tesla Custom".
You will need to provide an OAuth Access Token. You can generate one using tools like [MyTeslaMate](https://www.myteslamate.com/tesla-token/).
